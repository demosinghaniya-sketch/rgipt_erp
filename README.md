📘 RGIPT ERP Assistant

A smart, user-friendly platform to access and analyze RGIPT ERP data with enhanced features like attendance tracking, skip calculation, and a seamless UI — available as both a website and Android APK.

🚀 Features
📊 Attendance Tracking
View subject-wise attendance
Detailed insights into present/absent records
🎯 75% Criteria Calculator
Know how many classes you can skip
Know how many you must attend to maintain 75%
🔐 Captcha-Free Login
Uses a proxy-based system to simplify login experience
📱 Android APK Support
Installable app version for easy mobile access
🌐 Web Version
Accessible directly through browser
⚡ Fast & Interactive UI
Clean design for smooth navigation
🛠️ Tech Stack
Frontend:React
App: Android (WebView using Capacitor)
Hosting: Railway / Vercel
📥 Installation
🌐 Use Website
Simply open:
https://rgipt-erp.vercel.app/login

📱 Install APK
link (https://github.com/codeXlucky12/rgipt_erp/releases/download/apkv0.0.1/RGIPT-nexus-v1.0.apk)
Go to the Releases section of this repository
Download the .apk file
Install it on your Android device

⚠️ You may need to enable "Install from Unknown Sources"
⚙️ How It Works
The app connects to the official ERP system
Fetches attendance data
Processes it to:
Calculate percentage
Predict safe skips
Show detailed analytics

🔒 Security Note
This app does NOT store your credentials
All data is fetched securely during runtime
APK is completely safe to use
